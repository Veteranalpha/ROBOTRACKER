# ROBOTRACKER
A CAR BOT THAT CAN NAVIGATE ITS ENVIRONMENT AND  AVOID OBSTACLES BY ITSELF

COMPONENENT USED 
ARDUINO  UNO
L298N MOTOR DRIVER
2 -HC -SRO4 ULTRASONIC SENSOR
4 -DC MOTORS
2 -9V BATTERIES
MINI BREADBOARD
HOT GLUE
JUMPER WIRES
